Demonstration (API Call)

The API provides simple, secure endpoints for any data consumer (e.g., a frontend app, another service) to get curated data.


curl -X POST https://my-data-api.com/get_data \\
     -H "Authorization: Bearer ..." \\
     -d '{
       "dataset_name": "reviews_spec",
       "filters": {"city": "Paris", "year": 2023}
     }'

API JSON Response:


{
  "status": "success",
  "record_count": 1250,
  "data": [...]
}

Key Technical Decisions

  • Insight: API latency kept under 100ms for P90 requests.
  • Decision: Chose to query Iceberg via Athena/Presto for a "data serving layer" instead of a traditional DB, reducing data duplication.
  • Impact: Proves maturity in "data product thinking" and full integration of the data catalog.
